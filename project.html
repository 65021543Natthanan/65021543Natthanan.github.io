MemoRice - Android Application for Rice Disease Management

MemoRice is a mobile application designed to streamline the process of recording and managing rice field data for farmers. Recognizing the challenges faced by farmers in tracking crucial information related to their crops, particularly concerning diseases, this project aims to provide a user-friendly and efficient digital solution. MemoRice empowers farmers to systematically log field details, disease symptoms, visual evidence (images), and other relevant environmental factors. This facilitates better decision-making, improved disease management, and ultimately contributes to increased productivity and reduced losses. The application was developed with a focus on ease of use, accessibility, and data integrity.

- Use Flutter as the core technology for user interface (UI) development and core functionality of mobile applications on the Android operating system.
- Use Node.js and Express.js to develop backend systems and APIs for communication between mobile applications and databases.
- Use MySQL as a relational database management system (RDBMS) to store and manage application-related data.
